# support-miniapp-runtime

Biblioteca headless e agnostica de UI/UX para composicao minima entre crates transversais de mini-apps.

## Capacidades

- contexto minimo de app
- criacao de instancias documentais
- geracao de evento documental base
- integracao minima com numeracao, auditoria via `core-audit` e autoria

## Contrato

Consultar `MAN.md`.

## Testes

```bash
cargo test -p support-miniapp-runtime
```
