# support-versioning

Biblioteca headless para versionamento funcional local e release notes persistentes em JSON.

## Capacidades

- representacao de `ReleaseNotes`
- store em ficheiro com JSON formatado
- inicializacao automatica do ficheiro
- atualizacao de versao, novidades e problemas conhecidos

## Contrato

Consultar `MAN.md`.

## Testes

```bash
cargo test -p support-versioning
```
